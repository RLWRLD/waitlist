# Ralph Mode (끈기 모드)

## 입력
```
/ralph [작업 설명]
```

**예시**:
- `/ralph 전체 코드베이스를 TypeScript로 마이그레이션`
- `/ralph 모든 API 엔드포인트에 에러 핸들링 추가`
- `/ralph CI/CD 파이프라인 구축 완료까지`

## 핵심 원칙

### 절대 중단 금지
- 에러 발생 → 직접 디버깅, 다른 접근 시도
- 막히면 → 우회 방법 탐색, 부분 완료 후 계속
- "이 정도면 됐다" 금지 → 원래 요청이 100% 완료될 때까지

### 검증 없이 완료 선언 금지
완료를 선언하려면 반드시:
1. 완료를 증명하는 **명령어를 실행** (테스트, 빌드, 린트 등)
2. 출력 결과를 **직접 확인**
3. 모든 항목이 통과한 **증거 제시**

## 금지 표현
아래 표현이 나오면 검증이 안 된 것:
- "아마", "될 것 같다", "should work"
- "나머지는 비슷하게 하면 됩니다"
- 실행 없이 "완료했습니다"

## 작업 루프
```
1. 작업 항목 파악
2. 구현
3. 검증 명령 실행
4. 실패 시 → 수정 후 3번으로
5. 전체 통과 → 다음 항목 또는 최종 검증
6. 최종 검증 통과 → 완료 선언 + 증거 제시
```

## 규칙
- 한 번에 하나씩 확실하게 완료
- 검증 명령어는 반드시 실행 (추측 금지)
- 실패하면 원인 분석 후 재시도
- 부분 완료로 끝내지 않기
- 최종 결과에 검증 증거 포함
